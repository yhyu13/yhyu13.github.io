Key Findings

  • Finding 1: Agent Harness Engineering 被定义为第三代工程范式（继 Prompt Engineering → Context Engineering 之后），六组件架构为 Game AI 提供了可复现评估的基础设施。
  • Finding 2: 游戏场景是 Harness 测试的完美床——Minecraft 有清晰的物理规则、丰富的动作空间、可验证的结果，但创造性评估仍是开放问题。
  • Finding 3: 从"手工设计环境"→"自动生成环境"的演进趋势明确（Agent World Model），游戏引擎正从"环境"升级为"完整的 Agent 治理基础设施"。
